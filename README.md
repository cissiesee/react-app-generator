## 简介
react app 创建命令集

## 安装
### 全局安装
    npm install react-app-generator -g
### 本地项目全局链接（仅工具开发者关注）
    npm link

## 使用
**注：请在项目根目录下运行**
### 创建react项目
    react-app --dir <reactprojectname>
### 创建cotainer
    react-app --container <containername>
### 创建component
    react-app --component <componentname>
### 创建reducer
    react-app --reducer <reducername>
